# Sunny Kumar - Portfolio

A meticulously crafted, highly interactive, and visually stunning personal portfolio showcasing engineering precision and creative intuition. Built from the ground up to reflect a premium editorial aesthetic.

## 🚀 Live Demo
[View Live Portfolio](https://srs-portfolio.vercel.app/)

## ✨ Key Features

- **Custom Premium Aesthetic:** Engineered without heavy CSS frameworks. Relies on meticulously tuned vanilla CSS (`style.css`) with responsive fluid typography, glassmorphism, and subtle micro-animations.
- **Dynamic Theming:** Seamless Light and Dark mode transitions featuring an expanding circular ripple effect.
- **GSAP Scroll Animations:** High-performance hardware-accelerated scroll reveals and parallax hero effects powered by GSAP and Intersection Observers.
- **Interactive Lightbox Galleries:** Dual bespoke galleries (Film Strip & Masonry Grid) with custom 0.5s delayed lightboxes, providing immersive viewing for photography without interrupting navigation.
- **Vite & React Architecture:** Fully configured Vite project setup with TypeScript support, ready for future expansion or component hydration.

## 💻 Tech Stack
- **Structure:** Semantic HTML5
- **Styling:** Advanced Custom CSS (`style.css`)
- **Logic:** Vanilla ES6+ JavaScript (`main.js`)
- **Animations:** GSAP & ScrollTrigger
- **Build Tool:** Vite

## 🛠️ Local Development

1. **Clone the repository:**
   ```bash
   git clone https://github.com/em-srs/srs-portfolio.git
   cd srs-portfolio
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. **Build for production:**
   ```bash
   npm run build
   ```

## 📂 Project Structure
```text
/
├── index.html        # Main entry point and semantic structure
├── style.css         # All global styles, animations, and variables
├── main.js           # Core interactive logic (Lightboxes, GSAP, Theme toggle)
├── assets/           # UI elements, resumes, graphical assets
├── photos/           # Photography gallery assets
├── package.json      # Dependencies and build scripts
└── vite.config.ts    # Vite bundler configuration
```
